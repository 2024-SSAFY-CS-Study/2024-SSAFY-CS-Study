### 스터디 방식
- 매주 월요일에 한 사람당 한 주제를 정하여 정리하고 금요일까지 commit
- 3주마다 대면 스터디 진행: 3주간 내용을 바탕으로 질문에 답하는 방식


### commit convention
- 처음 올릴 때: `feat: Web/Http Status Code - 주효림`
- 이후 수정할 때: `fix: Web/Http Status Code - 주효림`
- README 수정할 때: `docs: Update READEME`
